# Galette des Rois Guide — Shop Directory for Japan

## Problem
Finding galette des rois in Japan is extremely hard. Information is scattered
across individual bakery Instagram posts and a few incomplete blog articles.
No comprehensive directory exists in Japanese OR English.

Owner personally struggled to find this info last year. Fans search obsessively
every December-January (#ガレットデロワ = 13,100+ Instagram posts/year).

## Solution
The most complete galette des rois shop directory in Japan. Bilingual (EN/JP).
Covers Tokyo, Osaka, Kyoto, Yokohama + online delivery.

### Core features
1. **Shop Directory** — 34+ shops with prices, reservation info, delivery
2. **Reservation Deadline Calendar** — when to order by (some shops need 2 weeks)
3. **Price Comparison** — all shops on one scale (¥2,500-¥9,000)
4. **Feve Gallery** — collectible figurines from each shop (photos)
5. **Home Baking Guide** — where to buy molds, frangipane, puff pastry in Japan
6. **Allergy Info** — nut-free options flagged

### Killer differentiators
- Most comprehensive list in any language
- Reservation deadline tracking = unique utility
- Owner visits patisseries for first-hand photos and tasting
- Feve collector angle (niche but passionate)
- Bilingual (reaches both expats and Japanese searchers)
